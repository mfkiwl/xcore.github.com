Firmware Overview¶

The module in this repo is a feasibility study on coding an EtherCAT slave on an XS1 device. The target is either an L2 or a G4, depending on whether fast threads are required (> 100 MIPS requires an L2) and on whether a fast switch is required (the G4 switch is very low latency).
